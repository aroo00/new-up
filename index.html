⚠ One-time Firebase Setup Needed

To share live data across all devices, connect a free Firebase database. Do this once — takes about 5 minutes:

1
Go to console.firebase.google.com → sign in with your Google account.
2
Add project → name it (e.g. 7eleven-rack) → Continue → Create project.
3
Left menu → Build → Realtime Database → Create Database → any location → Start in test mode → Enable.
4
⚙ Gear icon (top-left) → Project Settings → scroll to Your apps → </> (Web) → Register → copy the firebaseConfig.
5
Open this HTML file in Notepad/text editor → find 🔥 PASTE YOUR FIREBASE CONFIG HERE → replace each PASTE_..._HERE with your values → save.
6
Reopen the file. Admin logs in to check sections. Share the same link — viewers see everything live with no login! ✅
